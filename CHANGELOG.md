# Changelog

## How to Use This Changelog

As you move from top to bottom in the changelog, you'll encounter older versions. The version at the top represents the latest release, which includes the most recent features and improvements, while the versions further down contain earlier updates with fewer or older functionalities.

- **Unreleased**: Changes that are still being worked on or have not yet been included in a release.
- **Added**: New features or improvements added.
- **Changed**: Updates to existing features, refactoring, or improvements.
- **Fixed**: Bug fixes and patches.
